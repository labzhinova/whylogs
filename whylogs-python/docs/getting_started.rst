Getting Started
================

WhyLogs library comes with quickstart CLI to help you initialize the configuration. You can also use the API directly without \
going through the CLI.

Quick Start
===================

Install the Library
#####################
Install our library in a Python 3.6+ environment.

.. code-block:: bash

   pip install whylogs

#######################
